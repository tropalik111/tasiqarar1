## الجزء الأول: تنزيل الأدوات

### ١. تنزيل Node.js

١. اذهب إلى: **https://nodejs.org**
٢. اضغط على الزر الأخضر **"LTS"** (الإصدار الموصى به)
٣. حمّل الملف وثبّته (Next, Next, Install)

### ٢. تنزيل Git

**على ماك:**
- افتح Terminal (طبق "الطرفية")
- اكتب: `git --version` واضغط Enter
- إذا طلب منك التثبيت، اضغط "Install"

**على ويندوز:**
١. اذهب إلى: **https://git-scm.com/download/win**
٢. حمّل الملف وثبّته بالإعدادات الافتراضية

### ٣. تنزيل محرر النصوص (VS Code)

١. اذهب إلى: **https://code.visualstudio.com**
٢. حمّل وثبّت

## الجزء الثاني: إنشاء حساب GitHub

GitHub هو المكان الذي سيُخزَّن فيه الكود.

١. اذهب إلى: **https://github.com/signup**
٢. أدخل إيميلك واختر اسم مستخدم وكلمة مرور

## الجزء الثالث: إنشاء مستودع للموقع

١. بعد تسجيل الدخول في GitHub، اضغط على الزر الأخضر **"New"** (أو علامة `+` أعلى اليمين)
٢. املأ التفاصيل:
   - **Repository name:** `tasiqarar`
   - **Description:** Qarar - Saudi Market Intelligence Platform
   - اختر **Public**
   - ✅ **لا** تختر "Add a README file"
٣. اضغط **"Create repository"**

سترى صفحة بها أوامر — احتفظ بها مفتوحة، سنحتاجها بعد قليل.

## الجزء الرابع: تجهيز الكود محلياً

### ١. فك ضغط ملفات المشروع

١. حمّل ملفات المشروع التي أرسلتها لك
٢. فك الضغط (unzip) في مكان تتذكره — مثل: سطح المكتب
٣. ستجد مجلد اسمه `tasiqarar`

### ٢. فتح Terminal/PowerShell

**على ماك:**
- افتح Terminal
- اكتب `cd ~/Desktop/tasiqarar` (أو المسار الذي حفظت فيه المجلد)

**على ويندوز:**
- افتح PowerShell
- اكتب `cd C:\Users\اسمك\Desktop\tasiqarar`

أو **الطريقة الأسهل:** انقر بالزر الأيمن على مجلد `tasiqarar` واختر "Open in Terminal" (ماك) أو "Open PowerShell window here" (ويندوز).

### ٣. تثبيت الحزم

في الـ Terminal، اكتب:

```bash
npm install
```

اضغط Enter وانتظر دقيقة أو دقيقتين. ستظهر أرقام كثيرة — هذا طبيعي.

### ٤. تجربة الموقع محلياً (اختياري لكن موصى به)

اكتب:

```bash
npm run dev
```

سترى رسالة فيها رابط مثل: `http://localhost:5173`

افتح هذا الرابط في المتصفح — يجب أن ترى موقعك يعمل! 🎉

للوصول إلى لوحة الإدارة:
- اذهب إلى: `http://localhost:5173/#admin`
- كلمة المرور: `qarar2026`

**⚠️ مهم جداً:** قبل النشر، غيّر كلمة المرور في الملف `src/Qarar.jsx` السطر ١٤:

```javascript
const ADMIN_PASSWORD = "اكتب_كلمة_مرور_قوية_هنا";
```

اختر كلمة مرور قوية (مثلاً: حروف+أرقام+رموز، ١٢ حرف على الأقل).

لإيقاف السيرفر المحلي، اضغط `Ctrl + C` في الـ Terminal.

## الجزء الخامس: رفع الكود إلى GitHub

في الـ Terminal (داخل مجلد المشروع)، اكتب الأوامر التالية واحداً تلو الآخر:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
```

ثم انسخ السطرين التاليين من صفحة GitHub التي تركتها مفتوحة (سيبدآن بـ `git remote add origin...`)، أو استخدم هذه (غيّر `username` إلى اسم مستخدمك في GitHub):

```bash
git remote add origin https://github.com/username/tasiqarar.git
git push -u origin main
```

سيُطلب منك تسجيل الدخول إلى GitHub. اتبع التعليمات.

بعد الانتهاء، حدّث صفحة GitHub — ستجد كل ملفاتك مرفوعة! ✅

## الجزء السادس: النشر على Vercel

Vercel ستستضيف موقعك مجاناً.

### ١. إنشاء حساب Vercel

١. اذهب إلى: **https://vercel.com/signup**
٢. اضغط **"Continue with GitHub"** (الأسرع)
٣. وافق على الصلاحيات

### ٢. ربط المستودع

١. في لوحة Vercel، اضغط **"Add New..."** → **"Project"**
٢. ستجد قائمة بمستودعات GitHub الخاصة بك
٣. اضغط **"Import"** بجانب `tasiqarar`
٤. في صفحة الإعدادات:
   - **Framework Preset:** Vite (يجب أن يُكتشف تلقائياً)
   - اترك باقي الإعدادات كما هي
٥. اضغط **"Deploy"**

انتظر ١-٢ دقيقة... ☕

عند الانتهاء، ستحصل على رابط مؤقت مثل: `tasiqarar-xyz.vercel.app`

افتحه — موقعك شغّال على الإنترنت! 🚀

---

## الجزء السابع: ربط الدومين tasiqarar.com

### ١. في Vercel

١. في صفحة المشروع، اذهب إلى **"Settings"** → **"Domains"**
٢. اكتب: `tasiqarar.com`
٣. اضغط **"Add"**
٤. أضف أيضاً: `www.tasiqarar.com`

ستظهر لك تعليمات DNS — سننفّذها الآن.

### ٢. في موقع شراء الدومين

اذهب إلى لوحة تحكم الدومين (Namecheap, GoDaddy, أو أينما اشتريته):

١. ابحث عن إعدادات **"DNS"** أو **"Manage DNS"** لـ tasiqarar.com
٢. احذف أي سجلات A و CNAME موجودة (إن وجدت)
٣. أضف السجلات التالية:

| النوع | الاسم | القيمة |
|---|---|---|
| A | @ | 76.76.21.21 |
| CNAME | www | cname.vercel-dns.com |

٤. احفظ التغييرات

⏱️ **انتظر ١٠ دقائق إلى ساعتين** — انتشار DNS يحتاج وقت.

### ٣. التحقق

ارجع إلى Vercel → Settings → Domains. سترى علامة ✅ خضراء بجانب الدومين عندما يكون جاهزاً.

افتح **https://www.tasiqarar.com** — موقعك جاهز! 🎉

## استكشاف الأخطاء

### المشكلة: `npm install` لا يعمل
**الحل:** تأكد من تثبيت Node.js. أعد فتح Terminal بعد التثبيت.

### المشكلة: `git push` يطلب كلمة مرور باستمرار
**الحل:** استخدم **Personal Access Token** بدلاً من كلمة المرور:
١. GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
٢. Generate new token → اختر "repo"
٣. استخدمه ككلمة مرور

### المشكلة: Vercel يفشل في النشر
**الحل:** تأكد أن `package.json` و `vite.config.js` موجودان في المستودع.

### المشكلة: الدومين لا يعمل بعد ٢ ساعة
**الحل:** تحقق من DNS في:
- https://www.whatsmydns.net (اكتب tasiqarar.com)
- إذا لم تنتشر، انتظر ٢٤ ساعة قصوى
